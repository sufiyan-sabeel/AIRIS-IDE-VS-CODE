# AIRIS Mobile IDE

Mobile-first PWA IDE built with Next.js, Monaco Editor, and TypeScript. Optimized for Android browsers and PWA installation.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **Monaco Editor** - VS Code's editor engine
- **xterm.js** - Terminal emulator
- **Framer Motion** - Animations
- **Zustand** - State management

## Quick Start

```bash
git clone https://github.com/sufiyan-sabeel/AIRIS-IDE-VS-CODE.git
cd AIRIS-IDE-VS-CODE
npm install
npm run dev
```

Open http://localhost:3000 on your mobile browser.

## Build

```bash
npm run build
```
